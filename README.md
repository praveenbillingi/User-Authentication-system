# User-Authentication-System

A beginner-friendly user authentication system built using Python and Flask.

## Features
1. User Registration
2. User Login
3. Password Hashing
4. SQLite Database

## Tech Stack
1. Python
2. Flask
3. SQLite
4. HTML

## How to Run
1. Install Python
2. Install Flask
3. In terminal -> Run `python app.py`
